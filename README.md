# Student Management System with Calculator and To-Do List

This project is a Python-based Student Management System that combines three useful features into a single application.

The system not only stores and manages student records but also provides a calculator for performing calculations and a To-Do List for managing daily tasks. Instead of using separate applications, users can access all these features in one place.

## Features

* Add student records
* Search student details by roll number
* Display all student records
* Delete student records
* Perform mathematical calculations using a calculator
* Create and manage a To-Do List

This project demonstrates the use of Python concepts such as functions, lists, dictionaries, loops, and conditional statements. It is designed for beginners to practice CRUD operations and menu-driven programming.

The idea behind this project is "3 in 1" functionality: Student Management + Calculator + To-Do List, making it simple and useful for students to manage data, perform calculations, and organize tasks efficiently.
